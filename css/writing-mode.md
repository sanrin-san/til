## 関連するCSSプロパティ

### [`text-orientation`](https://developer.mozilla.org/ja/docs/Web/CSS/text-orientation)

> このプロパティは縦書きのテキスト (`writing-mode` が `horizontal-tb` 以外の場合) でのみ効果があります。

`mixed`: default  
`upright`: 英文等の横書き用の文字を90°回転しない
